This folder contains locally defined tscircuit component wrappers used by the boards in `lib/`.
Each entry is listed as a filename followed by a short description.

- imports/ACS37800.tsx
ACS37800 power monitor IC footprint wrapper.

- imports/ATTINY85V_10SU.tsx
ATTiny85 microcontroller in SOIC-8.

- imports/AVR_ISP_2x3.tsx
2x3 AVR ISP programming header footprint.

- imports/ISM330DHCX.tsx
ST ISM330DHCX 6-axis IMU (accelerometer/gyro) footprint.

- imports/JoystickThumb.tsx
Joystick module footprint with multiple pins.

- imports/MMC5983MA_QFN16.tsx
MMC5983MA 3-axis magnetometer QFN-16 footprint.

- imports/PinHeader1x3.tsx
1x3 pin header footprint.

- imports/PinHeader1x4.tsx
1x4 pin header footprint.

- imports/SM04B_SRSS_TB_LF__SN.tsx
JST SH 4-pin right-angle connector footprint.

- imports/TC78H670FTG_EL.tsx
Toshiba TC78H670FTG motor driver IC footprint.

- imports/VGF39NCHXT_B103.tsx
3-pin component footprint used in circuit02.
